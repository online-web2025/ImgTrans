##  3.此代码转化混合图片格式时候，图片名字不能重合，否则不能正常转化
##  4.此代码具有修改图片尺寸功能
##############################

##  jpg,png,bmp,webp -> png
##  jpg,png,bmp,webp -> bmp

##  jpg,png,bmp,webp -> jpg,png,bmp,webp
